# QTranser
一款简单的翻译软件

## 非常方便使用
您只需要按下Ctrl+C将文字复制进剪切板：

这是一款从Mac中叫快译的软件模仿来的

学代码肯定要读英文，我尝试了很多翻译软件

唯独快译是最方便使用的

几乎不会离手，使用率特别高

我想让windows用户也享受到如此的便利

于是从头学习C#将它实现出来。
